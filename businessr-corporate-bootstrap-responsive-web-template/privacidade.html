Política de Privacidade e Proteção de Dados

A TECiNOVA respeita sua privacidade e está comprometida com a proteção dos seus dados pessoais. Esta política explica como coletamos, usamos, armazenamos e compartilhamos suas informações, conforme os princípios da Lei Geral de Proteção de Dados (LGPD) – Lei nº 13.709/2018.

1. Coleta e Uso de Dados Pessoais

Coletamos apenas informações pessoais essenciais para fornecer nossos serviços, como nome, e-mail, telefone e dados de navegação.

2. Bases Legais para o Tratamento de Dados

  • Consentimento do titular para envio de comunicações.
  • Execução de contrato, quando necessário para prestar serviços.
  • Cumprimento de obrigação legal, como armazenamento de registros.
  • Legítimo interesse, para análise e melhoria da plataforma.

3. Retenção e Proteção dos Dados

Os dados são armazenados conforme padrões de segurança para evitar acessos indevidos ou vazamentos.

4. Direitos do Titular dos Dados

Você pode acessar, corrigir, excluir ou solicitar a portabilidade dos seus dados através do e-mail contato@tecinova.top.

5. Compartilhamento de Dados

Seus dados não serão vendidos e só serão compartilhados quando necessário para cumprir obrigações legais ou operacionais.

6. Política de Cookies

Utilizamos cookies para melhorar sua experiência. Você pode gerenciá-los diretamente nas configurações do seu navegador.
