Política de Privacidade e Proteção de Dados

A TECiNOVA respeita sua privacidade e está comprometida com a proteção dos seus dados pessoais.

1. Dados Coletados e Finalidade

  • Dados fornecidos pelo usuário: Nome, e-mail, telefone, endereço.
  • Dados coletados automaticamente: IP, cookies, histórico de navegação.

A coleta ocorre para:

  • Melhorar a experiência do usuário.
  • Cumprir obrigações legais.
  • Garantir a segurança da plataforma.

2. Bases Legais para o Tratamento de Dados

  • Consentimento do titular para envio de comunicações.
  • Execução de contrato para prestação de serviços.
  • Cumprimento de obrigação legal, como armazenamento de registros.
  • Legítimo interesse para análise e melhorias.

3. Retenção e Proteção dos Dados

Os dados são armazenados conforme padrões de segurança para evitar acessos indevidos.

4. Direitos do Titular dos Dados

  • Confirmação da existência de tratamento
  • Acesso aos dados coletados
  • Correção ou exclusão dos dados
  • Portabilidade dos dados
  • Revogação do consentimento

Solicitações podem ser feitas pelo e-mail contato@tecinova.top.

5. Compartilhamento de Dados

Seus dados não serão vendidos e só serão compartilhados quando necessário.

6. Transferência Internacional de Dados

Se houver necessidade de transferência internacional, garantimos conformidade com a LGPD.

7. Política de Cookies

Utilizamos cookies para melhorar sua experiência. Você pode gerenciá-los nas configurações do navegador.

8. Alterações na Política de Privacidade

Esta política pode ser atualizada periodicamente.
